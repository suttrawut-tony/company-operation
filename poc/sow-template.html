2 บทสรุปผู้บริหาร (Executive Summary)

บริษัท ศาลาแดง จำกัด ("SLD") มีความยินดีนำเสนอขอบเขตการทำงานสำหรับโครงการวางระบบ {{SAP Business One / Oracle NetSuite}} ให้กับ {{ชื่อบริษัทลูกค้า}} ("ลูกค้า") โดยโครงการนี้มีวัตถุประสงค์เพื่อ{{สรุปเป้าหมายหลัก เช่น ยกระดับระบบ ERP ให้ครบวงจร ครอบคลุมงานบัญชี-การเงิน จัดซื้อ ขาย และคลังสินค้า}}

ระบบ ERP จะช่วยให้ลูกค้าสามารถ{{ระบุ value proposition หลัก เช่น ลดขั้นตอนงานซ้ำซ้อน เพิ่มความถูกต้องของข้อมูลทางการเงิน}} ทั้งนี้ SLD จะดำเนินการตามมาตรฐาน Implementation Methodology ที่ผ่านการพิสูจน์จากโครงการกว่า 50 องค์กรในประเทศไทยและภูมิภาค

8 สมมติฐานโครงการ (Project Assumptions)

สมมติฐานเหล่านี้เป็นพื้นฐานในการประเมินขอบเขตและค่าใช้จ่าย หากเปลี่ยนแปลง อาจส่งผลต่อ timeline และค่าใช้จ่าย
  1. ระยะเวลา SOW (SOW Term): บริการมีผลภายใน 12 เดือนนับจากวันออกใบแจ้งหนี้
  2. ข้อจำกัดขอบเขต (Scope Limitation): จำกัดเฉพาะ Section 4.1 งานนอกขอบเขตผ่าน Change Request
  3. การให้บริการ (Service Delivery): ดำเนินการแบบ Remote {{ยกเว้นกรณี Onsite}}
  4. ความพร้อมลูกค้า (Customer Resources): ลูกค้าจัด Key User และผู้มีอำนาจตัดสินใจเข้าร่วมตามกำหนด
  5. คุณภาพข้อมูล (Data Quality): ลูกค้ารับผิดชอบความถูกต้องของ Master Data ที่จัดเตรียม
  6. การยกเลิก/เลื่อนนัด (Cancellation): ยกเลิกซ้ำโดยไม่แจ้งล่วงหน้าอาจมีค่าใช้จ่ายเพิ่ม
  7. รูปแบบรายงาน (Report Templates): ใช้ Template มาตรฐาน SLD ปรับแต่งเพิ่มเป็นงานนอก Scope
  8. จำนวนฐานข้อมูล (Database): จำกัด {{1}} Production + {{1}} Test
  9. ปริมาณข้อมูล (Data Volume): เกิน limit ที่ระบุ SLD สงวนสิทธิ์ออก Change Order
  10. ชำระเงิน (Payment): ชำระก่อน Project Kick-Off ตาม Section 10
